# 🍳 Recipr – AI Recipe Platform

> Turn your leftovers into masterpieces using AI 🤖✨

Recipr is a **full-stack AI-powered recipe platform** that helps users generate delicious recipes from available ingredients or pantry images — reducing food waste and making cooking smarter.

---

## 🌟 Live Demo
🔗 https://recipr-one.vercel.app


---

## 🚀 Key Features

### 🤖 AI Powered
- Generate recipes using ingredients
- Smart cooking suggestions
- AI-based pantry scanning (image → ingredients)

### 🧑‍🍳 Cooking Experience
- Step-by-step cooking guide
- Cooking time, servings, and nutrition info
- Smart recommendations (e.g., Butter Chicken, Brownies)

### 📦 Pantry Management
- Scan pantry using camera 📸
- Add items manually
- Track ingredients easily

### ❤️ Personalization
- Save favorite recipes
- "My Recipes" dashboard
- Category & cuisine browsing

### 📄 Utility Features
- Download recipes as PDF
- Clean and modern UI
- Fast and responsive design
